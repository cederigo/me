The images are split into the three sets
- trainingset (tr)
- validationset (va)
- testset (te)

The files are in the following format
X-Y-Z.tif

where X  in {N,J} and indicate the class
Y in [1,21] and indicate the writer id
Z in [1,10] and indicate the image nr

N-11-5.tif is therefore the 5th 'N' written by writer nr. 11
